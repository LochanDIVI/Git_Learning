
========================================
Untracked :new files that git doesn't yet track

Modified :changed that git doesn't yet track

Staged :file is ready to be committed

Unmodified :unchanged after execute commit

Git commands

clone & Status

Clone -cloning a repository on our local machine
git clone <https git repository link>

Status -displays the state of the code
git status

git add <file>
git add .

git commit -m " "

git push origin main

git init

git remote add origin <link>

git remote -v (to verify remote)

git branch (to check branch)
git branch -M main (to change master to main)
git checkout -b <new branch name> (to create a new branch)
git checkout <branch name> (to change from one branch to another)
git branch -d <branch name> (to delete the branch)
